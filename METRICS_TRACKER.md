## Interpretation notes
- Stars are a lagging signal, not the only success metric.
- If visitors are high but stars are low, improve README clarity, demo, install path, and value proposition.
- If visitors are low, improve channel targeting and post usefulness.
- If comments ask the same question, update README.
- If people request a feature, add it to the roadmap and consider a follow-up post after shipping it.
