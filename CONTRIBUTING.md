# Contributing

Thanks for helping improve `oss-launch-lint`.

## Local setup

```bash
npm install
npm test
npm run lint
npm run build
```

## Pull request checklist

- Keep checks practical for maintainers and small projects.
- Add or update tests for behavior changes.
- Keep README examples accurate.
- Do not add features that fabricate, buy, automate, exchange, or incentivize stars.
- Do not require GitHub tokens, paid APIs, or account login for the default audit.

## Good issues to open

- A launch-readiness check produced noisy advice.
- A repo type is unsupported or scored unfairly.
- The generated report is unclear.
- Installation or quickstart instructions failed from a clean checkout.
