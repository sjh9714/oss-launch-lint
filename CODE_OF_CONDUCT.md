# Code of Conduct

This project is for practical, respectful collaboration.

## Expected behavior

- Be kind and direct.
- Assume good intent while making room for correction.
- Keep feedback focused on the work.
- Respect maintainers' time and community rules.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Spam, star manipulation, fake engagement, or unsolicited bulk promotion.
- Sharing private information, credentials, or security details publicly.

## Reporting

If something needs maintainer attention, open an issue or use the private security reporting path described in [SECURITY.md](SECURITY.md) when appropriate.
