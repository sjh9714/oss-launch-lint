# Security Policy

## Supported versions

`oss-launch-lint` is currently pre-1.0. Security fixes will target the latest release.

## Reporting a vulnerability

Please do not open a public issue for vulnerabilities.

Until GitHub private vulnerability reporting is enabled for the repository, contact the maintainer through the repository owner's preferred private channel and include:

- Affected version or commit
- Reproduction steps
- Impact
- Suggested fix, if known

Do not include secrets, tokens, private repo contents, or personal data in reports.
