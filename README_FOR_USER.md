# Codex star-ready repo pack

This pack gives Codex enough persistent context to build and ethically launch a small GitHub repository that has a realistic path to 16+ legitimate stars.

## How to use
1. Create a new local folder or GitHub repo.
2. Copy all files in this pack into the repo root.
3. Edit `GOAL_PROMPT.md` placeholders.
4. Start Codex in the repo.
5. Paste the `/goal` prompt from `GOAL_PROMPT.md`.
6. Approve only safe actions: dependency installs, release creation, package publishing, and external posts.
7. Do the external posts manually if Codex cannot access your accounts.
8. Track results in `METRICS_TRACKER.md`.

## Recommended default repo idea
Use `oss-launch-lint` unless you already have a sharper idea. It is meta, useful, easy to demo, and directly aligned with the launch goal without becoming a star-manipulation tool.

## Most important instruction
Do not optimize for fake engagement. Optimize for a tiny useful tool, fast onboarding, clear README, and targeted feedback-driven promotion.
